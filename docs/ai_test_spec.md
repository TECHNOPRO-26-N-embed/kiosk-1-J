## 5. 主な不具合・設計差分
1. CSVファイル名と参照先がモジュール間で不一致
	- 例: `books.csv` / `ai_book.csv`、`users.csv` / `ai_users.csv`、`history.csv` / `ai_history.csv`
2. 貸出ログと返却処理の履歴フォーマットが不一致
	- 貸出側は `CHECKOUT,user_id,book_id,date` 形式
	- 返却側は「貸出状況: ..., ユーザーID: ...」形式
3. 会員管理更新処理の書式不備
	- `fprintf` の `%*[^,]` 指定が不適切で、CSV更新の正当性を満たせない
